# nodejs_rtsp
 simple mockup for live streaming test


## Documentation


## Run Locally

1. Clone the project

```bash
  git clone https://github.com/RaisHelmy/nodejs_rtsp.git
```

2. Go to the project directory

```bash
  cd nodejs_rtsp
```

3. Install dependencies

### Windows:
```bash
  choco install ffmpeg
```
### Windows:
```bash
  apt install ffmpeg
```

4. Start the server

```bash
  node rtsp_server.js  
```

5. Open client file named "rtsp_client.html"

```bash
  rtsp_client.html
```
